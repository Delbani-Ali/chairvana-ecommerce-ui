# 🪑 Chairvana – Furniture E-Commerce UI

A modern **furniture e-commerce front-end** built using HTML and CSS.
Focused on clean layout, product presentation, and interactive UI effects.

---

## 📌 Overview

Chairvana is a static e-commerce interface showcasing furniture products.
The project emphasizes **visual hierarchy, hover interactions, and structured sections** similar to real online stores.

No backend. No frameworks. Pure front-end work.

---

## 🚀 Features

* Clean and structured navigation bar (categories + icons)
* Full-width hero banner
* Category cards with hover effects
* Promotional posters with animation
* Quote/testimonial section
* Service highlights (delivery, support, etc.)
* Product grid with:

  * ⭐ Ratings
  * 💰 Pricing
  * 🔄 Flip card animation (front/back images)
* Interactive footer with social icons
